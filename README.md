Kanban Węgrzynkiewicz
=====================

Zwykły projekt na zaliczenie zadań labolatoryjnych z wykorzystaniem Symfony

## Instalacja

```
$ composer install
$ php bin/console doctrine:schema:update
$ mysql -h localhost -P 3306 -u root -p kanban < kanban.sql
```
